Admin Module:

1) Add Movie Hall to the system:

addHall(HallJSON):
input: HallJSON
HallJSON will contain : hallName,street,city,state,zipcode,screen_nums.
response: hallId

2) Add Movie to the system

addMovie(MovieJSON)

input: MovieJSON
MovieJSON will contain: movieName,description,seeItIn,trailer,photos,cast,movie_length,release_date,genres
response: movieId

3) addScreen(screenJSON)

input: screenJSON
screenJSON will contain: hallName,screen_num,screen_type,total_seats
response: screenId

4) getMovies(search_movie)

input: search_movie (It can be substring of a movie_name)
response: list of MovieJSON's
MovieJSON will contain: movieId,movie_name,description,seeItIn,trailer,photos,cast,movie_length,release_date,genres

5) getHalls(search_hall)

input: search_hall(It can be a substring of a hall_name)
response: list of HallJSON's
HallJSON will contain: hallId,hall_name,street,city,state,zipcode,screen_nums

6) getScreens(hallId)
input: hallId
response: list of screenJSON
screenJSON will contain: screenId,hallId,screen_num,screen_type,total_seats

7) editMovie(MovieJSON)
input: MovieJSON
MovieJSON will contain: movieId,movie_name,description,seeItIn,trailer,photos,cast,movie_length,release_date,genres
response: UpdatedMovieJSON
UpdatedMovieJSON will contain : movieId,movie_name,description,seeItIn,trailer,photos,cast,movie_length,release_date,genres

8) editHall(HallJSON)
HallJSON will contain: halId,hall_name,street,city,state,zipcode,screen_nums
response: updatedHallJSON
updatedHallJSON: hallId,hall_name,street,city,state,zipcode,screen_nums

9) editScreen(ScreenJSON)
screenJSON will contain: screenId,hallId,screen_num,screen_type,total_seats
response: updatedScreenJSON
updatedScreenJSON will contain: screenId,hallId,screen_num,screen_type,total_seats

10) getBills(bill_date / bill_month)
input: bill_date / bill_month
response: list of billJSON
billJSON will contain: billId,userId,movie_scheduleId,amount,tax,total_price,no_of_seats,booking_date.

11) getBillByID(billId)
input: billId
response billJSON
billJSON will contain: billId,userId,movie_scheduleId,amount,tax,total_price,no_of_seats,booking_date.
